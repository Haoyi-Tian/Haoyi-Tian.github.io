General Projects

This page is a collection of various projects not related to flight. Many projects involve a combination of several methods and tools, while some are focused on specifically one (such as Arduino or MATLAB).

RC Airplane Design and Build Project I

This project was initiated out of personal interest and the desire to fly a remote controlled airplane. Everything was originally designed and built from scratch except for the electrical components (motor, battery, etc). This was the first attempt at making an RC airplane, so the primary goal was to learn some basics about building and to achieve somewhat controlled flight. The project page is still a work in progress.

Cam-Guided Laser Image Tracing

The goal of this project was to design a pair of cams for a laser "lightshow" setup. A laser pointer rests on the two cams, its position dependent on the varying radii of the cams. A desired path for the laser was first specified before the cams were designed and fabricated based on this desired laser path.

User-Controlled Lifting Vehicle

This robot was a team project for the robot contest in MAE3 (Introduction to Engineering Graphics and Design). The goal was to build a user-controlled robot that could transport blocks from various locations on the playing field and deposit them in a designated area to score points.

Finite Element Analysis of Truss

This MATLAB project uses finite element analysis to characterize a 2D truss shaped like a power transmission-line tower. Varying loads can be applied to the two overhanging arms to examine the deformation of each member and to visualize the truss under load.

Thermometer

This Arduino project uses a thermistor to calculate temperature and display it on an LCD panel in both Celcius and Fahrenheit scales. The main goal of this project was to practice using the LCD display, while the thermistor was included to add some additional functionality and turn it into a thermometer.
